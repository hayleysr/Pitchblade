// reyna 
// Tooltips for buttons and controls

// top bar buttons
presetButton	= Open and manage your presets
bypassButton	= Toggle all effects on or off
settingsButton	= Open plugin settings

//indivdual daisy chain buttons
modeButton	 = Change chaining mode
mode.down    = Down: Sends audio to the next effect
mode.split   = Split: Sends audio to two effects at the same time
mode.double  = Double Down: Runs two effects in parallel
mode.unite   = Unite: Merges two parallel effects together
bypass		= Bypass this effect temporarily

// global daisy chain buttons
addButton		= Add a new effect to the chain
duplicateButton = Duplicate an existing effect
deleteButton	= Remove an effect from the chain

// effect discriptions
effect.Gain			= Adjusts overall volume level
effect.Noise Gate	= Controls background noise by muting quiet sounds
effect.Compressor	= Reduces dynamic range to even out loud and quiet parts
effect.Formant		= Shifts vowel/tone characteristics without changing pitch
effect.Pitch		= Detects pitch to the nearest target note
effect.De-Esser		= Reduces harsh, high-frequency 's', 'sh', and 't' sounds (sibilance)
effect.De-Noiser    = Removes background noise
effect.Equalizer	= Adjusts balance of frequency components

//each effect panels textboxs/sliders/buttons
effect.Gain.level			= Adjusts the output volume level
effect.Noise Gate.threshold	= Sets the level below which audio is muted
effect.Noise Gate.attack	= Sets how quickly the gate opens when audio exceeds the threshold
effect.Noise Gate.release	= Sets how quickly the gate closes when audio falls below the threshold
effect.Compressor.threshold	= Sets the level above which compression is applied
effect.Compressor.ratio		= Sets the amount of compression applied to audio above the threshold
effect.Compressor.attack	= Sets how quickly compression is applied when audio exceeds the threshold
effect.Compressor.release	= Sets how quickly compression is released when audio falls below the threshold
effect.Formant.shift		= Adjusts the amount of formant shifting applied
effect.Pitch.correction		= Sets the strength of pitch correction applied
effect.Pitch.speed			= Sets how quickly pitch correction is applied
effect.De-Esser.threshold	= Sets the level above which de-essing is applied
effect.De-Esser.frequency	= Sets the frequency range targeted for de-essing
effect.De-Noiser.threshold	= Sets the level above which noise reduction is applied
effect.De-Noiser.reduction	= Sets the amount of noise reduction applied
effect.Equalizer.lowGain	= Adjusts the gain for low frequencies
effect.Equalizer.midGain	= Adjusts the gain for mid frequencies
effect.Equalizer.highGain	= Adjusts the gain for high frequencies
